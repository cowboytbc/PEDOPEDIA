## How to Access Documents (Step by Step)

### Using CourtListener (FREE):

1. Go to https://www.courtlistener.com/
2. Search for: "Giuffre v. Maxwell" OR "Jeffrey Epstein"
3. Click on the case docket
4. Browse "DOCKET ENTRIES" tab
5. Look for entries with PDF icons
6. Click "Download PDF" on documents of interest
7. Many are free, some redirect to PACER

### Using PACER (Small Fee):

1. Create account at: https://pacer.uscourts.gov/
2. Register for CM/ECF access
3. Search by case number or party name
4. Download documents (charged per page)
5. Fee waivers available if under $30/quarter

## IMPORTANT: Verification Guidelines

When collecting documents for your search tool:

1. **Verify Source**: Only use documents from:
   - Official court websites
   - CourtListener/PACER
   - Government websites
   - Major news organizations

2. **Check Document Stamps**: Official documents should have:
   - Court seals
   - Case numbers
   - File stamps
   - Attorney names

3. **Cross-Reference**: 
   - Compare with multiple sources
   - Verify case numbers match
   - Check dates for consistency

4. **Avoid**:
   - Social media posts
   - Unverified websites
   - Screenshots without sources
   - "Leaked" documents without provenance
